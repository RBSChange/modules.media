Gestion des propriétés d’un document

Une fois le document intégré dans la médiathèque, double-cliquer sur son titre. La fenêtre des propriétés s’affiche :

Les champs obligatoires sont marqués par . Les champs non-valides sont marqués par l’icône .

  • Définir les champs suivants :
    • « Intitulé complet » : Nom du document sous Change.
    • « Contenu alternatif court » : Le contenu alternatif court est limité à 60 caractères pour des raisons d’accessibilité. Le texte inséré dans ce champ s’affichera sous la forme d'une info bulle lors d’un survol souris sur l’image.
    • « Contenu alternatif long » : Champ textuel permettant de donner des informations complémentaires sur le document avec moins de limitation quant au nombre de caractères utilisés.
    • « Crédit » : Champ textuel libre pour indiquer les crédits liés à une image par exemple.
    • : Bouton permettant de faire s'ouvrir une fenêtre pour la recherche d'un nouveau fichier sur son disque dur et de mettre à jour l'ancien document.

Une fois l’ensemble des propriétés du document indiquées, possibilité est offerte de cliquer sur :

  • : Pour sauvegarder les informations
  • : Pour vider le contenu des champs et recommencer la saisie
  • : Pour enregistrer les propriétés du document tout en fermant ensuite la fenêtre des propriétés

Pour supprimer un document :

  • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
  • Cliquer sur

Pour mettre à jour un document (sans éditer ses préférences) :

  • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
  • Cliquer sur

Une fenêtre s'ouvre permettant de parcourir son disque dur et de mettre à jour le document.

Pour récupérer un fichier et l'enregistrer sur son disque dur :

  • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
  • Cliquer sur

Une fenêtre s'ouvre permettant d'ouvrir le document ou de le sauvegarder sur son disque dur.

Il est possible d’ordonner ensuite les différents documents insérés :

  • Sélectionner le répertoire dans lequel ils ont été intégrés (sur la gauche de l’applicatif)
  • Cliquer sur
  • Utiliser les flèches afin de faire remonter ou de faire redescendre un document dans la liste
  • Cliquer sur pour valider

Pour visualiser une miniature d’un document préalablement intégré dans la médiathèque :

  • Sélectionner le document en cliquant sur son titre au centre de l’applicatif
  • Cliquer sur la touche F1 du clavier
