Gestion des documents

Le module « Médiathèque » permet de stocker les images, les PDF et éventuellement les animations Flash s’intitule « Médiathèque ».

Des répertoires peuvent être créés afin de classer les documents par thème.

Pour créer un nouveau répertoire :

  • Sélectionner le dossier racine ‎
  • Cliquer sur .

Les propriétés du répertoire s’affichent alors au milieu de l’applicatif :

  • Renseigner le champ obligatoire (« Libellé » est l’intitulé du dossier) signalé par(si ce champ est mal renseigné, il ne sera pas valide et il ne sera pas possible d’enregistrer les propriétés du répertoire. Cela sera signalé par).
  • Cliquer sur pour sauvegarder les modifications.

: Permet de créer d’autres répertoires (prendre garde à bien enregistrer les propriétés du répertoire précédent avant…).

Pour insérer un document dans la médiathèque :

  • Ouvrir ce module
  • Sélectionner le dossier dans lequel on souhaite insérer le fichier
  • Glisser-déposer le document depuis le disque dur dans la médiathèque sous Change :

La liste des documents se recharge.
